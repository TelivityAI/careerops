# Security Policy

## Supported versions

The `main` branch of this repository is the supported line for the open-source dashboard.

## Reporting a vulnerability

Please **do not** file a public GitHub issue for security problems.

1. Open a [GitHub Security advisory](https://github.com/TelivityAI/careerops/security/advisories/new) on this repo, or  
2. Contact a TelivityAI organization owner on GitHub.

Include steps to reproduce and impact. We will acknowledge and work on a fix as quickly as we can.

## Please never publish

- Supabase service-role keys or other secret keys  
- Other people’s resumes, emails, or job data  
- Credentials for the hosted demo at careerops.telivity.app
